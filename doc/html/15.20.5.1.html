15.20.5.1  Variable
  - This defines a local variable of the form 
     TD_V_name
which  contains the value that you have defined.  You may define variables
before entering TOPDRAWER so that they are available for use.  You do this
by:
     $ TD_V_NAME := "number"
                                  example
     $ TD_V_A := "25.8E5"
     $ TOPDRAWER
     $ TITLE TOP 'A=',T_A
