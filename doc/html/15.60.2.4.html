15.60.2.4  Point_example
  - To Read in an array of 3 dimensional data 
     READ MESH
               Y  y1 y2 .... yn
          X x1 Z z11 z21 ... zn1
          X x2 Z z12 z22 ... zn2
          ...............
          X xm Z z1m z2m ... znm

or...
     READ MESH
          FOR      X= x1 x2 .... xn
          FOR Y=y1 Z=z11 z21 ... zn1
          FOR Y=y2 Z=z12 z22 ... zn2
          ...............
          FOR Y=ym Z=z1m z2m ... znm
