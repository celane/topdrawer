5.3  OPTIONS
 

     1.  /[NO]COMMAND[=file]  -  Specifies  a  file  to  get  commands for
         TOPDRAWER setup.
         (Default:TDINIT.TOP)

     2.  /[NO]JOURNAL[=file] - Specifies a file to journal all interactive 
         commands.
         (Default:TD.TDJ)
     3.  /[NO]LIST[=file]  -  Specifies  a  file  to list the commands and
         error messages.
         (Default:TD.LIS)
